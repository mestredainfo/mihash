# MIHash

O MIHash, abreviação para "Verificador de Hash", é uma ferramenta projetada para assegurar a integridade de arquivos baixados da Internet. Funciona verificando se o hash fornecido pelo desenvolvedor ou fornecedor coincide com o hash gerado localmente a partir do arquivo baixado. Isso proporciona uma camada adicional de segurança, garantindo que o arquivo não tenha sido corrompido durante o download ou alterado por terceiros.

## Executando da Fonte

Para executar MIHash a partir do código-fonte, siga estas etapas:

- Extraia o arquivo zip
- Instale as dependencias necessárias: `npm install`
- Inicie o aplicativo: `npm start`
